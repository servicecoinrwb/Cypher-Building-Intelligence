Deployment Logic

Three distinct operating modes to handle the lifecycle of a project.

01
/// CONSTRUCTION PHASE

Install Mode

Silent operation. Power up the panel during construction. Outputs are mechanically locked out. No nuisance alarms. No relay clicking. Safe for electricians.

02
/// SALES PHASE

Demo Mode

The ultimate sales tool. Disconnect sensors and the Physics Engine takes over, simulating real temperature reactions on the dashboard to close the deal in the boardroom.

03
/// PRODUCTION PHASE

Live Mode

Full PID control. Safety interlocks active. If a sensor wire is cut, Cypher executes an emergency safe-shutdown immediately to protect the equipment.
