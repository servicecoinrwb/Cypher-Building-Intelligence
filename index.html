Architecture

How it’s deployed in the real world.

SC-1

SC-1 Supervisory Controller

The building hub. Aggregates RTUs, schedules occupancy, hosts the dashboard, and centralizes trend/log views.

Typical SC-1 functions
  • • Facility schedule → publishes occupied/unoccupied points
  • • Live dashboards + operator overrides
  • • Trend snapshots (as configured per site)
  • • Integrates to Modbus devices (relays/AO/UI boards)
FC-1

FC-1 Field Controller

Mounted per unit (RTU/AHU). Handles local I/O, safety logic, staging, and publishes points to the MQTT backbone.

Typical FC-1 functions
  • • Local heat/cool staging + fan control
  • • Demand control ventilation (CO₂) if equipped
  • • Sensor normalization + fault flags
  • • Continues running even if SC-1 is offline
Protocol/Stack (Current Reality)
Node-RED • Mosquitto MQTT • Modbus RTU • (Optional) Zigbee2MQTT • Web UI
