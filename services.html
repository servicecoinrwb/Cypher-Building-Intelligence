CYPHER OPEN BAS Request Proposal
Engineering • Install • Commissioning

From Scope to Startup.
Done right.

Cypher installs are built like real controls projects: clean panels, documented points, verified I/O, and operator training. No fluff — just systems that work and are serviceable.

01

Scope + Point List

Planning & Documentation

We start with equipment scope (RTUs/AHUs/exhaust/lighting), required control points, and a clean point list. This defines sensors, outputs, network layout, and what the dashboard will expose to the operator.

  • Equipment inventory + control goals
  • Point list + I/O mapping
  • Network plan (MQTT/Modbus)
02

Panel Build (Shop)

Clean Fabrication

Panels are built clean and serviceable. SC-1 (supervisory) and FC-1 (field) controllers are mounted and labeled. We terminate I/O cleanly and prep the communication bus (RS-485 / Ethernet) so field work is faster and cleaner.

  • DIN rail, wire duct, labeled terminals
  • SC-1 / FC-1 integration + IO-88 expansions
  • 24VAC control power + fused distribution
03

Field Install

On-Site Execution

We mount panels, land field wiring, and install sensors. Communications are brought online and verified end-to-end. This step focuses on clean terminations and correct signal types — so commissioning is predictable.

  • Panel mounting + field terminations
  • RS-485/Ethernet network bring-up
  • Sensor placement + verification
04

Commissioning + Training

Checkout & Handoff

We do a point-to-point checkout, validate sequences, and confirm safeties. Then we train staff on the dashboard: schedules, overrides, alarms (if used), and how to verify the system is operating correctly.

  • Point checkout (inputs/outputs)
  • Sequence validation (heat/cool/fan/occ)
  • Operator training + documentation handoff
